# Task Management System

Система управления задачами с API, веб‑интерфейсом и десктоп‑приложением.

## Обзор

Проект реализует полнофункциональную систему управления задачами (Task Management System) с использованием современных технологий. Система включает:

- API для работы с задачами;
    
- веб‑интерфейс через Minimal API;
    
- десктопное приложение на WPF; (разработка в планах)
    
- асинхронную обработку задач через RabbitMQ;
    
- кэширование в Redis;
    
- хранение данных в PostgreSQL.
    

## Архитектура

### 1. Основной API (ASP.NET Core с Minimal API)

Отвечает за:

- CRUD‑операции с задачами;
    
- аутентификацию пользователей;
    
- работу с проектами и подзадачами.
    

### 2. База данных (PostgreSQL с EF Core)

Реализует:

- таблицы для пользователей, проектов, задач;

- отношения между сущностями;
    
- миграции для управления схемой данных.
    

### 3. Автомаппинг (AutoMapper)

Обеспечивает:

- преобразование DTO в сущности и обратно;
    
- минимизацию boilerplate‑кода.
    

### 4. RabbitMQ

Используется для:

- отправки уведомлений;
    
- ~~обработки отложенных задач;~~
    
- ~~синхронизации данных.~~
    

### 5. Redis

Применяется для:

- ~~кэширования часто используемых данных;~~
    
- хранилища сессий;
    
- ~~счётчиков и статистики.~~
    

### 6. WPF‑клиент (разработка в планах)

Включает:

- интерфейс для работы с задачами;
    
- ~~оффлайн‑режим с синхронизацией;~~
    
- ~~систему уведомлений.~~
    

## Технологии

- **Backend**: ASP.NET Core (Minimal API).
    
- **База данных**: PostgreSQL + Entity Framework Core.
    
- **Автомаппинг**: AutoMapper.
    
- **Очереди сообщений**: RabbitMQ.
    
- **Кэширование**: Redis.
    
- **Десктоп‑клиент**: WPF (.NET).
